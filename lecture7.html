1. What is DataTypes?

Data type humko ye batata hai ki hum variables me kis type ka data rakhne wale hai

Data Types represent ki hum kis type ki value store kar rhe hai apne variables me

Example of Data Types are: number(1, 2, 3), string(abc), boolean(T/F)

(DataTypes) humko batata hai ki "variables" kis type ki "value" store kar ne wala hai like, number, string, boolean etc.

(variables) basically aik container hai, jo ki apne andar koi value store karega.

Javascript is loosely type(dynamic type) programming language.

Humko javascript me variable ka type khud se batane ki jarurat nhi hai

2. Types of DataTypes

Primitive Data Types

    Memory Types different types ke hota hai Pehla hai (Stack)

    Stack
  1. number
  2. string
  3. boolean
  4. null
  5. undefined
  6. symbols (newly introduced) go on mozilla website to learn more

Reference Data Types

    Memory Types different types ke hota hai dusra hai (Heap)

    Heap
  1. array
  2. object
  3. date
  4. function
  5. set
  6. map and many more.

Reference Data Types(RDT): Aisee datatype jo allocate hote hai, yaa jo banate hai (Heap) memory mai. (Heap) hamare "RAM" ke memory ka small parts hai.
  • (Heap) data type dynamic hota hai and fixed nhi hota hai, "Not Fixed"
  • jo bhi hum variable baneye gye, agar voo reference type ka hai too voo (Heap) me declare hoga, (Heap) wali memory use karega.
  • Yaha pe hum "object" banate hai, ji ki fixed nhi hoti hai.

    • Primitive Data Types(PDT): ye mool deta type hote hai, ji ki declare aur banate hai (Stack) memory mai.
  • (Stack) ka matlab "Fixed" hota hai
  • yaha jo bhi hum variable declare karege voo (Stack) me jayega.
