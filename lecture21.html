Dom Manipulation in JavaScript

What is DOM

DOM(document object Model) it is platform and language independent model to represent the html or xml documents.

DOM ki help se hum (html and xml document) ko represent kar sakte hai.

What is the Work of DOM?

it defines the logical structure of the documents and the way in which they can be accessed and manipulated by an application program.

  • Example are: DOM humko tree like structure hamare document ko provide kar dega, jisase hum apne elements ko easily access and manipulate kar paaye inside our program.
    • In the DOM, all parts of the document, such as elements, attributes, text etc are organized in the hierarchical tree-like structure.
  • Like Parent Node, child Node, siblings.
    • In DOM terminology these individual parts of the document ar known as "nodes".
